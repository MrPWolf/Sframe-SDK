# Contributing to Sframe SDK

We love your help! Sframe is an Open Core project. While the SDK is open-source under Apache 2.0, the hosted dashboard and advanced automation features are part of our SaaS.

## How to Contribute

1.  **Report Bugs**: Open an issue on GitHub.
2.  **Suggest Features**: We are always looking for new "Interceptor" or "Sentinel" logic ideas.
3.  **Pull Requests**: 
    - Fork the repo.
    - Create a feature branch.
    - Ensure `npm run lint` passes.
    - Submit a PR!

## Development

- `bin/init.js`: The CLI installer logic.
- `src/templates/`: The core engine components (Sentinel & Interceptor).
- `schema.json`: The manifest validation schema.
